## Overview

Create a USD or NGN virtual card for a customer. This endpoint allows you to issue a new virtual card with a specified currency, brand, and initial funding amount. A wallet_id can be optionally specified to change the default currency funding wallet. The card will be linked to the customer's account and can be used for online and in-store purchases.
